Set boundary zone:
Region Parameters
Boundary zone
Specify a region; any station (in the input file) falling outside will be omitted. The region should be given as a rectangle, specifying min/max values in longtitude and latitude (using decimal degrees). E.g. 21.0/23.5/36.0/38.5
Grid Step
The x-axis and y-axis grid step size in degrees. This option is only relevant if the program computes more than one strain tensors. Default is 1 (deg).
